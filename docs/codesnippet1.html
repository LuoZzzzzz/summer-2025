Distributing Balls in Bins

Suppose for whatever reason we want to get all the possible ways of distributing $k$ balls in $n$ bins. The following small piece of code accomplishes it recursively in a few simple lines. 

def helper(remaining: int, classes_left: int, prefix: List[int]):
      if classes_left == 1:
        counts.append(prefix + [remaining])
        return
      for i in range(remaining + 1):
        helper(remaining - i, classes_left - 1, prefix + [i])

The above function helper(k, n, []) returns a list of all possible ways to distribute the k balls among the n bins.

The recursion tree looks like this for $k=2$, $n=3$: 

(2,3,[])
--------(2,2,[0])
-----------------(2,1,[0,0]) #2-0=2
----------------------------append([0,0,2])
-----------------(1,1,[0,1]) #2-1=1
----------------------------append([0,1,1])
-----------------(0,1,[0,2]) #2-2=0
----------------------------append([0,2,0])
--------(1,2,[1])
-----------------(1,1,[1,0]) #1-0=1
----------------------------append([1,0,1])
-----------------(0,1,[1,1]) #1-1=0
----------------------------append([1,1,0])
--------(0,2,[2])
-----------------(0,1,[2,0]) #0-0=0
----------------------------append([2,0,0])
